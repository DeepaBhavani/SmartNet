-> SmartNet Campus 
AI-Driven WiFi Quality Monitoring & Optimization System  
(GDG Circuit Sprint Hackathon – MVP)



-> Problem Statement
Campus WiFi is often weak due to poor coverage .
Currently, issues are reported informally.


-> Solution
SmartNet conceptually monitors campus WiFi and provides:
- Dead zone detection
- Heatmap visualization
- AI-based insights
- Admin dashboard analytics


-> MVP Screens 
1. **Student Screen:** Shows WiFi strength and status  
2. **Admin Dashboard:** Heatmap of campus WiFi  
3. **AI Insights:** Gemini-powered recommendations  



-> Google Technologies
- Firebase – Real-time storage  
- Google Maps API – Heatmap visualization  
- BigQuery – Analytics  
- Gemini API – AI predictions  
- Cloud Functions – Automated alerts  


->Future Scope
- Router-level WiFi integration  
- Predictive outage prevention  
- Hostel / library specific analytics
